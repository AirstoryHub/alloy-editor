Alloy Editor

Editing made sexy again!

Alloy Editor is a modern WYSIWYG editor built on top of CKEditor, designed to create modern and gorgeous web content.

Works on IE7+, Chrome, Firefox and Safari.

3 - Start using AlloyEditor

Load the module, initialize AlloyEditor and specify the configuration.

YUI().use('alloy-editor', function(Y) {
    var editor = new Y.AlloyEditor({
        srcNode: '#editable',
        toolbars: {
            add: ['image', 'code'],
            image: ['left', 'right'],
            styles: ['strong', 'em', 'u', 'h1', 'h2', 'a', 'twitter']
        }
    });
});

And that is everything you need to get started!
