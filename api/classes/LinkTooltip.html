    LinkTooltip Class

    Defined in: src/ui/yui/src/plugins/linktooltip.js:11

    The LinkTooltip class provides functionality for showing tooltip over link elements in editor. The class will be also registered as CKEDITOR plugin - CKEDITOR.plugins.linktooltip
