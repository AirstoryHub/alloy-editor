Play with the content, don't struggle with it.

AlloyEditor - WYSIWYG Editor, build on top of CKEditor. By Liferay.

Editing the content on the fly

Yes, you can edit this content right here and right now. Just select some text and the toolbar for styling will appear. Not only it will appear, but it will appear where you expect it - where you release the mouse. If you don't have mouse, but only keyboard, the toolbar will appear where selection starts, or ends.

Paste content

You can also paste content directly from web! In most cases, it will work out of the box.

Images aligning

Oh, and we also support images aligning.

Drag&Drop images

Want to Drag&Drop images from your Desktop directly to the editor? Nothing easier than this. Try now right here (but keep in mind this requires IE10+, Chrome, Firefox or Safari).

Styling content

Currently we have implementation of the following styles:

  • Strong (Bold)
  • Emphasize (Italic)
  • Underline
  • Heading Level 1
  • Heading Level 2

Adding content

Editing is good, but what about adding content? Well, right now you can:

  • Insert images
  • Add links
  • Send tweets

Browser support

AlloyEditor supports IE7+, Chrome, Firefox and Safari.

P.S. Do you know that H4 button, which you probably used to add/remove Headings Level 4 (H4 styling), like "Editing the content on the fly" was especially added for this page? It does not exists as default button in AlloyEditor. However, adding a new button is extremely easy. Here is the code of H4 button:

YUI.add('button-h4', function(Y) {
        'use strict';

        var Lang = Y.Lang;

        var H4 = Y.Base.create('h4', Y.Plugin.Base, [Y.ButtonBase], {
             // There is no icon for H4 right now, but look ma, even a simple text is enough.
             // In production, it would be better to add icon instead raw text.
            TPL_CONTENT: '<i class="alloy-editor-icon-h4">h4</i>'
        }, {
            NAME: 'h4',

            NS: 'h4',

            ATTRS: {
                element: {
                    validator: Lang.isString,
                    value: 'h4'
                }
            }
        });

        Y.ButtonH4 = H4;

    }, '', {
        requires: ['button-base']
    });

And then we used it just like this:

YUI().use('alloy-editor', 'button-h4', function(Y) {
        var editor = new Y.AlloyEditor({
            srcNode: '#contentEditable',
            toolbars: {
                add: ['image'],
                image: ['left', 'right'],
                styles: ['strong', 'em', 'u', 'h1', 'h2', 'h4', 'a', 'twitter']
            }
        });
    });
