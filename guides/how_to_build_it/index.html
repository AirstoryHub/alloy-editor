How to build it?

How to build a version of AlloyEditor.

1 - Install NodeJS

Using your browser, navigate to http://nodejs.org and install NodeJS.

2 - Install Gulp

Open a terminal in your computer and execute 
[sudo] npm install -g gulp

3 - Fork AlloyEditor

Use the __Fork__ button in our [Github Repository page](https://github.com/liferay/alloy-editor) to get a copy of alloyEditor. After that, follow the instructions to clone the project into your computer.

4 - Install AlloyEditor dependencies

Navigate to the AlloyEditor project in your computer and execute 
[sudo] npm install

5 - Build AlloyEditor

Navigate to the AlloyEditor project in your computer and execute 
gulp build

There are other gulp tasks you can use like __release__ to generate a bundled zip file, or __watch__ if you're in development mode
