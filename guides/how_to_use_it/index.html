How to use it?

The section below assumes that you are using the default UI, which is built using React. The AlloyEditor core is fully separated from the UI and creating another one based on any other framework is very easy.

1 - Include AlloyEditor's CSS in the page

<link href="alloy-editor/assets/alloy-editor-ocean.css" rel="stylesheet"></link>

2 - Include AlloyEditor's JS in the page

There are a few ways to include the editor on the page:
Add the whole editor with all UI on the page 
<script src="alloy-editor/alloy-editor-all.js"></script>
Add it on the page separating the CKEditor code from AlloyEditor 
<script src="alloy-editor/ckeditor.js"></script>
<script src="alloy-editor/alloy-editor-no-ckeditor.js"></script>
Add it on the page separating the React code from AlloyEditor 
<script src="alloy-editor/react.js"></script>
<script src="alloy-editor/alloy-editor-no-react.js"></script>
Add it on the page separating both the React and the CKEditor code from AlloyEditor 
<script src="alloy-editor/ckeditor.js"></script>
<script src="alloy-editor/react.js"></script>
<script src="alloy-editor/alloy-editor-core.js"></script>

3 - Invoke the static editable method of AlloyEditor

AlloyEditor.editable('myContentEditable');

Please, check our Toolbar Configuration Guide and the Features List to know how to customize your editor and get the most out of it!
